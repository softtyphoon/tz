
    主要的爬取对象是主页：www.firstp2p.com，主页一般不会放爬虫
    间隔30-60秒对主页爬取一次；
    每次爬取之后，会对每个项目的信息进行对比，当：
        项目情况没有变化，不会对项目页面进行爬取；
        项目余额变化超过总额的10%时，会进行爬取；
        剩余额度小于总额的20%时，会进行爬取；
        剩余时间小于30分钟时，会进行爬取；
        否则都不会进行爬取；
    
    运行，还是昨晚的方法进行程序运行。
    
    
    windows下步骤：
      1. 下载python 2.7进行安装
      2. 下载http://www.crummy.com/software/BeautifulSoup/bs4/download/4.0/，下载源代码
      3. 在命令行下，切换路径到解压后源代码的目录
      4. 执行安装命令：python setup.py install
      5. 在命令行下切换到程序源码的路径下，执行 python firstp2p.py
